Exercícios HTML + CSS | Aula 02

Com base nos exercícios anteriores:
Utilize o arquivo index.html para fazer o seguinte:

  1. Crie vários links externos que sejam abertos em novas abas e redirecionem ao:
    • Twitter
    • Facebook
    • Instagram
  2. Crie um link local que redirecione a um arquivo chamado hobbies.html
    • O que acontece quando clicamos nesse link?
  3. Crie um link de e-mail que permita "enviar" um e-mail ao seu endereço de e-mail pessoal com o assunto "Mensagem do site".
    Dica: para escolher o assunto do email basta usar ?subject=mensagem após o endereço de e-mail
  4. Crie um link interno (âncora) que permita ir até a sessão "Experiência Profissional"
    Exemplo: "Ver Experiência Profissional".
    Dica: Se seu site estiver sem barra de rolagem então crie textos para que seu site tenha barra de rolagem, você pode usar este Gerador de textos.
  5. Levando em conta o item b:
    • Crie um novo documento HTML chamado hobbies.html e salvá-lo na mesma pasta que o arquivo index.html.
    • Crie um título principal para o hobbies.html chamado "Meus Passatempos"
    • Vincule o arquivo CSS que você já tem criado com o hobbies.html
    • Crie um link local para ir do arquivo hobbies.html até o arquivo index.html

Chegou a hora de usar imagens, vídeos e áudios!
Comece criando uma pasta chamada images dentro da mesma pasta onde estão seus arquivos HTML

  1. Procure e baixe os ícones do: Twitter, Facebook e Instagram, após isso basta salvá-los na pasta images.
    Dica: Você pode procurar as imagens clicando aqui
  2. Transforme as imagens em links, use as tags criadas no item 1.
    Dica: Que tal colocar a tag de imagem dentro da tag de link?
  3. No Google Imagens, procurar e baixar pelo menos 3 imagens relacionadas a passatempos do seu interesse e salvá-las na pasta images.
    Exemplo: leitura, esportes, viagens, etc.
    Atenção:As fotos devem ter no máximo 1200px de largura
  4. Coloque as imagens do item anterior no seu arquivo hobbies.html.
    • Todas as imagens devem ficar com 360px de largura e altura deve ser proporcional
    • Todas devem ter uma descrição atribuída
  5. Escolha um vídeo do youtube e faça download dele em formato webm, após isso crie uma pasta chamada videos ela deve ficar junto com seus arquivos HTML.
    Dica: Você pode usar este serviço par fazer download do seu vídeo
  6. Crie um subtítulo chamado "Vídeo em HTML5"
  7. Coloque o vídeo no seu arquivo hobbies.html, ele deve ter:
    • 640px de largura
    • Controles de reprodução
    • Imagem de capa (de sua escolha)
  8. Escolha mais um vídeo do youtube e faça download dele em formato mp3, após isso crie uma pasta chamada audios ela deve ficar junto com seus arquivos HTML.
    Dica: Você pode usar este serviço par fazer download do seu áudio
  9. Crie um subtítulo chamado "Áudio em HTML5"
  10. Coloque o áudio no seu arquivo hobbies.html, ele deve ter:
    • Controles de reprodução
    • E um loop na reprodução

Agora vamos usar um pouco de CSS no hobbies.html
Vamos continuar usando o arquivo estilos.css

  1. O primeiro passo é escolher uma imagem que sirva como plano de fundo e baixá-la.
    Dica: Você pode usar este serviço
  2. Crie uma regra para colocar a imagem como plano de fundo, essa imagem não deve se movimentar quando a página rolar.
  3. Crie uma regra par o título principal que aplique uma cor de fundo em formato RGBA.
    Lembrete: Não esqueça de usar a transparência
  4. Ainda na mesma regra, atribua uma cor de fonte que contraste com a cor de fundo para que o texto fique legível.
  5. Crie uma regra usando uma classe chamada destaque que será atribuída a pelo menos 4 parágrafosde sua escolha, esta regra deve ter:
    • Uma cor de fundo e cor de fonte à sua escolha
      Lembrete: O texto deve ficar legível
    • A fonte deve ter 22px.
    • Se for necessário ajuste o espaçamento entre linhas de acordo com o tamanho da fonte.
