Exercícios HTML + CSS | Aula 03

Utilizando o mesmo arquivo (index.html) dos exercícios anteriores:

  1. Insira várias tags <strong> dentro dos textos mais importantes do CV.
  2. Insira várias tags <em> dentro dos textos que queremos destacar no CV.
  3. Insira pelo menos duas tags <mark> no CV.
  4. Inserir uma abreviatura <abbr> em alguma parte do CV.
    Lembrete: Relembrar o uso do atributo title.
  5. Usando CSS, mudar como quiser a aparência dos <strong>, <em> e <mark> para que eles fiquem diferentes.

Agora, vamos trabalhar com formulários. Para isso:
Comece criando um arquivo contato.html

  1. Dentro dele, insira um título principal que diga: "Formulário de Contato" o arquivo deve conter links que levem ao arquivo index.html e ao hobbies.html.
  2. Inserir um formulário que contenha os seguintes campos (inputs), com as respectivas labels:
    • Nome e sobrenome
    • E-mail
    • Assunto da mensagem (combobox suspenso com opções à sua escolha)
    • Gênero (radio button para as opções "feminino", "masculino" e "outros")
    • Passatempos (no mínimo 3 checkboxes, à sua escolha)
    • Comentários (textarea)
    • Botões de enviar e excluir
    Lembrete: Os campos nome e sobrenome, e-mail e comentários deverão ser obrigatórios.

Vamos para o CSS
Utilizando o arquivo estilo.css criado nas aulas anteriores:

  1. Criar uma regra que permita mudar o visual (pode ser o fundo) de todos os campos do formulário apenas quando eles estiverem em foco ("focus").
  2. Criar uma regra que permita redimensionar o campo comentários somente na vertical.
  3. Criar uma regra que permita alterar a cor, espessura da fonte e decoração dos links apenas quando o usuário passar o mouse sobre eles ("hover").
  4. Criar uma regra que permita alterar o visual das imagens do arquivo hobbies.html apenas quando o usuário passar o mouse sobre elas.
    Dica: Você pode usar esta declaração para testar: border-radius: 20px.
